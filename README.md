# @audio/amp

> Amp + cabinet simulation (Ableton Amp/Cabinet, Logic Amp Designer class).

| Package | What |
|---|---|
| `@audio/amp-tube` | tube preamp/power-amp stage |
| `@audio/amp-cabinet` | speaker cab — IR convolution, or a biquad-chain approximation without an IR |

Classical modeling only (waveshaping / WDF); neural amp modeling (NAM) is ML-tier — deferred with the rest of the ML lane.
